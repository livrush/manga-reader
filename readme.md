# Milk Book

![asdf](logo.svg)

Milk Book is a simple, desktop application for comic reading. Built with Electron and AngularJS.

---

## Current Features

* Import files to library
* Read files from library
* Save last read page

## Future Features

* Search library
* Update theme

## Current Accepted File Formats

* .zip
* .cbz
* .pdf

## Future File Formats

* .rar
